= 확장을 완료합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
두 노드가 클러스터에 결합된 후 AutoSupport를 구성하고 SP 네트워크를 완료하여 새로 추가된 노드 구성을 완료해야 합니다. 그런 다음, 확장된 클러스터를 확인하고 AutoSupport 메시지를 생성하여 확장을 완료합니다. 클러스터에서 SAN을 사용하는 경우 LUN 경로를 업데이트해야 합니다.
